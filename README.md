# Hospital Data Pipeline using PySpark

## Overview

This project is a data engineering pipeline built using PySpark. The pipeline processes hospital datasets, cleans the data, and performs transformations for analytics and reporting.

---

## Technologies Used

* PySpark
* Python
* SQL
* Pandas
* GitHub

---

## Dataset

Hospital healthcare dataset containing:

* Patients
* Doctors
* Appointments
* Admissions
* Billing
* Treatments

---

## Pipeline Flow

1. Load raw hospital data
2. Process data using PySpark
3. Clean and transform datasets
4. Store processed data
5. Run analytics using SQL

---

## Transformations

* Remove null values
* Remove duplicates
* Data cleaning
* Standardize columns
* Data type conversion

---

## Sample Analytics

* Total Patients
* Department-wise Admissions
* Billing Analysis
* Doctor Performance
* Treatment Trends
